## HTML的结构

```html
<!DOCTYPE html>
<html>
<head>
  <meta charset="utf-8" />
  <title>文档标题</title>
</head>
<body>

</body>
</html>
```

### 标签tag

标签以“`<`​”开始，以“`>`​”结束，

#### 起始标签

<title>文档标题</title>

#### 结束标签

结束标签中含有“`/`​”，

### 元素element

元素由起始标签、内容和结束标签构成，是HTML的基本单位。

<title>文档标题</title>

#### 属性

​`<meta charset="UTF-8">`​中的charset就是meta的属性之一。属性可自定义。

#### 值

​`<meta charset="UTF-8">`​中=号后的内容，即`"UTF-8"`​就是属性对应的值。
